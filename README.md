## Sobre mim

Desenvolvedor web com foco em **TypeScript** e **JavaScript**, construindo desde sistemas de ponto de venda até extensões de browser e jogos. Gosto de transformar ideias em produtos funcionais — seja uma interface de mercado, uma ferramenta de automação ou um projeto open source.

- Desenvolvendo meu [portfólio pessoal](https://github.com/adriak007/Meu-Portifolio)
- Trabalhando em **[Mercadinho](https://pdvkerocaixa.vercel.app)** — sistema de PDV em TypeScript
- Contribuindo para **[foe-helfer-extension](https://github.com/adriak007/foe-helfer-extension)** — extensão para Chromium e Firefox
- Explorando **Tailwind CSS** e boas práticas de front-end moderno
